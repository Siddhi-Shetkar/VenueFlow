# 🏫 Hall Availability App

A simple Flutter application to manage and track hall bookings based on date and time.

---

## ✨ Features

* 📅 Select date for booking
* 🏢 Choose different halls (Assembly Hall, Seminar Hall, Auditorium)
* ⏰ Pick start and end time
* ➕ Add new bookings
* 🗑️ Delete bookings
* 📋 View all bookings for selected hall and date

---

## 🛠️ Tech Stack

* Flutter
* Dart

---

## 📂 Project Structure

```
lib/
 └── main.dart
```

---

## 📌 How It Works

* Bookings are stored temporarily using a map structure
* Each booking is linked with a selected date and hall
* Users can add and remove bookings dynamically
